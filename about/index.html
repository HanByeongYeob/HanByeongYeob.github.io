MarkDown 사용법

게시물 등록 방법

  1. _post 폴더에 작성하기 원하는 태그 폴더에 파일을 만든다.
    ex) _post > it2024
  2. 파일명은 다음 규칙에 따른다.
    년도-월-일-태그명-타이틀명.md
    ex) 2024-05-24-it2024-title.md

  3. md 파일 상단에는 다음 내용을 필수 작성 후 MarkDown 작성법에 따라 게시글을 작성한다.

    layout: post
current: post
cover:  assets/images/welcome.jpg
navigation: True
title: 게시글 제목
date: 2024-05-31 09:00:00 +0900
tags: [it2024]
class: post-template
subclass: 'post'
author: hby


    • layout, current, cover, navigation, class, subclass 변경 X
    • title : 게시글 제목
    • date : 게시글 작성 날짜
    • tags : 원하는 태그
    • author : 작성자(자신의 이니셜)



MarkDown 작성법

  1. 제목 (Headings)
    제목은 #을 사용하여 작성합니다. #의 개수에 따라 제목의 수준이 달라집니다. 
    # 제목 1
## 제목 2
### 제목 3
#### 제목 4
##### 제목 5
##### 제목 6

#

  1. 강조 (Emphasis)
    텍스트를 굵게 또는 기울임으로 강조할 수 있습니다. 
    **굵은 글씨** 또는 __굵은 글씨__
*기울임 글씨* 또는 _기울임 글씨_
***굵고 기울임 글씨*** 또는 ___고 기울임 글씨___

#

  1. 리스트 (Lists)
    순서 있는 리스트와 순서 없는 리스트를 작성할 수 있습니다. ```
  2. 첫 번째 아이템
  3. 두 번째 아이템
  4. 세 번째 아이템
  • 첫 번째 아이템
  • 두 번째 아이템

  • 세 번째 아이템

  • 첫 번째 아이템
  • 두 번째 아이템
  • 세 번째 아이템 ```

#

  1. 링크 (Links)
    링크를 삽입할 때는 대괄호와 괄호를 사용합니다. 
    [링크 텍스트](https://www.example.com)

#

  1. 이미지 (Images)
    이미지를 삽입할 때는 링크와 비슷하지만, 앞에 !를 붙입니다. 
    ![대체 텍스트](https://www.example.com/image.jpg)

#

  1. 코드 (Code)
    한 줄의 코드는 백틱()으로 감쌉니다. 여러 줄의 코드는 세 개의 백틱(\)으로 감쌉니다. 
    `한 줄 코드`
\한 줄 코드\
 ```여러줄 코드```

#

  1. 블록 인용 (Blockquotes)
    블록 인용은 >를 사용합니다. 
    > 이것은 인용입니다.
> 
>  여러 줄의 인용입니다.

#

  1. 수평선 (Horizontal Rules)
    수평선은 —, ***, ___ 등을 사용합니다. 
    ---
***
___

#

  1. 표 (Tables)
    표는 |와 -를 사용합니다. 
    | 헤더 1 | 헤더 2 |
| ------ | ------ |
| 셀 1   | 셀 2   |
| 셀 3   | 셀 4   |

#

  1. 체크리스트 (Task Lists)
    체크리스트는 - [ ] (미완료)와 - [x] (완료)를 사용합니다. 
    - [ ] 할 일 1
- [x] 할 일 2
- [ ] 할 일 3
